## Project Overview
This project presents an interactive COVID-19 Data Analysis Dashboard built using Power BI. The dashboard provides insights into total COVID-19 cases, death rate, recovery rate, vaccination progress, and regional impact across countries and continents.

## Tools Used
- Power BI
- Microsoft Excel
- GitHub

## Key KPIs
- **Total Cases:** 1786M
- **Death Rate:** 0.01
- **Recovery Rate:** 0.99

## Dashboard Visuals
- Total COVID-19 cases by month (trend analysis)
- Country-wise total cases comparison
- Continent-wise distribution of COVID-19 cases
- Vaccination analysis by country
- Total deaths comparison across locations

## Insights from Dashboard
- COVID-19 cases peaked during March and remained high in subsequent months
- USA and India reported the highest number of total cases
- Asia and North America contributed the largest share of global cases
- Countries with higher vaccination numbers showed controlled death rates
- Recovery rate is significantly higher compared to death rate globally

## Dataset Description
The dataset includes:
- Date / Month
- Location (Country)
- Continent
- Total Cases
- Total Deaths
- Total Vaccinations
- Recovery Metrics

The data is sourced from publicly available COVID-19 datasets and cleaned for analysis.
